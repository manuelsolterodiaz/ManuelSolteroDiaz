# Proyecto ASIR – IES Horizonte  
**Autor:** Manuel Soltero Díaz  
**Ciclo:** CFGS Administración de Sistemas Informáticos en Red  
**Curso:** 2025/2026  

Este repositorio contiene todo el material desarrollado para mi proyecto final del ciclo ASIR.  
Incluye la infraestructura completa del IES Horizonte, desplegada con Vagrant/VirtualBox, así como los servicios de red, seguridad, base de datos y automatización.

---

## Contenido del repositorio

- **Infraestructura virtualizada** con Vagrant y VirtualBox  
- **Servicios de red**: DHCP, DNS, FTP, HTTP y balanceo de carga  
- **Reglas de firewall** con iptables  
- **Base de datos MariaDB**: diseño ER, modelo relacional y SQL  
- **Scripts de automatización** en Bash y PowerShell  
- **Aplicación Python** con interfaz gráfica (Tkinter)  
- **Pruebas y anexos**
- **Aplicación Web PHP** Sistema de Gestión de Notas
